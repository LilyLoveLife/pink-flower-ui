
## Installation


```
npm install --save pink-flower-ui

```

```
yarn add --save pink-flower-ui

```

```
pnpm add --save pink-flower-ui

```

## Examples

```
import React from 'react';
import { Button } from 'pink-flower-ui';

const App = () => (
  <div className="App">
    <Button type="primary">Button</Button>
  </div>
);

export default App;

```
